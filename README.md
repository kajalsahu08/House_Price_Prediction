# 🏠 House Price Prediction

A Machine Learning web app that predicts California house prices using XGBoost with 82.32% accuracy.

## 🔗 Live Demo
👉 https://house-price-prediction-zmm2.onrender.com

## 📸 Preview
> Enter property details like location, rooms, income etc. and get instant price prediction!

## 🛠️ Tech Stack
- **Backend** — Python, Flask
- **ML Libraries** — Scikit-learn, XGBoost, Pandas, NumPy
- **Frontend** — HTML, CSS, JavaScript
- **Deployment** — Render
- **Notebook** — Jupyter

## 📊 Model Comparison
| Model | R² Score | Accuracy | MAE |
|---|---|---|---|
| 🏆 XGBoost | 0.8232 | 82.32% | 31,554 |
| Random Forest | 0.8056 | 80.56% | 32,628 |
| Gradient Boosting | 0.7774 | 77.74% | 36,998 |
| Linear Regression | 0.5791 | 57.91% | 51,942 |
